首页的登录账号和密码都为：admin  这个项目是基于python+flask完成的，全部的逻辑代码在app.py中，HTML全部在templates文件夹中，没有对项目进行分层处理
该项目的主要功能为：
  用户可以上传一段视频，然后对该视频解析为一帧一帧的图片，并且保存到指定的文件夹下
  可以对上传的视频生成：基准标签
  可以将处理好的视频，通过视频流的形式在前端进行可视化的处理
